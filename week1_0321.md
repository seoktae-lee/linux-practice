# Week 1 - 0321 리눅스 실습

## 배운 명령어
- cat 파일명 : 파일 전체 내용 출력
- head -n 파일명 : 파일 앞에서 n줄 출력
- tail -n 파일명 : 파일 뒤에서 n줄 출력
- grep "단어" 파일명 : 파일에서 특정 단어 포함된 줄 검색
- find 경로 -name "*.확장자" : 특정 확장자 파일 찾기

## 실습 내용
- week1_0319.md 파일로 cat/head/tail 실습
- grep으로 ls, mkdir 키워드 검색
- find로 dev 폴더 내 .md, .py 파일 전체 검색
