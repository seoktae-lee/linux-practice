# Week 2 - 0323 리눅스 실습

## 배운 명령어
- brew list : 설치된 패키지 목록 확인
- brew install 패키지명 : 패키지 설치
- brew search 패키지명 : 패키지 검색
- tree 경로 : 폴더 구조 시각적으로 출력
- top : 실시간 프로세스 모니터링 (q로 종료)
- ps aux : 현재 실행 중인 프로세스 전체 목록

## 실습 내용
- brew install tree 로 tree 설치
- tree ~/dev 로 전체 프로젝트 구조 확인
- top 으로 실시간 CPU/메모리 사용량 확인

## top 화면 주요 항목
- PID : 프로세스 ID
- %CPU : CPU 사용률
- MEM : 메모리 사용량
- STATE : sleeping(대기) / running(실행 중)
