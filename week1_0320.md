# Week 1 - 0320 리눅스 실습

## 배운 명령어
- ls -la : 숨긴 파일 + 권한 전체 보기
- cd /절대경로 : 루트부터 전체 경로로 이동
- cd ~/경로 : ~ 는 /Users/leeseoktae 와 동일
- cd .. : 상위 폴더로 이동
- cd ../.. : 두 단계 위로 이동
- chmod 755 파일명 : 실행 권한 부여

## 권한 숫자 의미
- 7 = rwx (읽기4 + 쓰기2 + 실행1)
- 5 = r-x (읽기4 + 실행1)
- chmod 전: -rw-r--r--
- chmod 후: -rwxr-xr-x

## 맥 루트(/) 주요 폴더
- Users : 사용자 홈 디렉토리
- bin : 기본 명령어 모음
- etc : 시스템 설정 파일
- tmp : 임시 파일 (재부팅시 삭제)
